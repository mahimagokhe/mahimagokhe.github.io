✅ Weather App

Technologies: Angular, D3.js, Node.js

A responsive weather dashboard built with Angular for a modular UI, D3.js for interactive visualizations, and a lightweight Node.js backend for API caching. The app provides real-time forecasts, historic trend charts, and alerting for severe weather — designed to help users quickly understand current conditions and short-term trends.

Key Features

  • Real-time weather and hourly forecasts with a performant REST proxy (Node.js).
  • Interactive D3 visualizations for temperature, precipitation, and wind trends.
  • Geolocation & search with cached results for fast UX and reduced API calls.
  • Responsive Angular components with accessible UI and offline-friendly caching.

My Role & Implementation Notes

Implemented core Angular components, created D3 chart integrations, optimized data handling with RxJS and change-detection strategies, and built a Node.js caching layer to minimize external API requests.
