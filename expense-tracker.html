📊 Expense Tracker

Technologies: Angular, AI, Firebase, Material-UI

This project is an intelligent expense tracker with AI-powered spending classification, budget forecasting, and automated receipt parsing. It helps users monitor expenses, identify saving opportunities, and stay on top of monthly budgets through an interactive, responsive dashboard.

Key Features

  • AI-driven spend categorization and trend visualization
  • Budget recommendation engine with forecast alerts
  • Firebase authentication and real-time expense sync
  • Responsive Angular dashboard with Material Design
